# HRMS2 UAT Checklist

## Public Reporter

- เปิดหน้ารายงานผ่านรหัสผ่านกลางได้
- กรอกและบันทึกรายงานแบบ clinical ได้
- กรอกและบันทึกรายงานแบบ non-clinical ได้
- แนบไฟล์ที่อนุญาตได้
- ระบบคำนวณเวลาล่าช้าจากวันที่เกิดเหตุและวันที่รายงานได้

## Admin

- เข้าระบบและเห็นคิวรายงานทั้งหมด
- เปิดรายละเอียดรายงานและแก้ไขข้อมูลเบื้องต้นได้
- เปลี่ยนระดับความรุนแรงและเกิด severity history
- ส่ง 1 รายงานไปหลายทีมนำได้ และได้เลขรันแยกกัน
- ตั้งค่าปีงบประมาณและ active fiscal year ได้
- ตั้งค่า visibility และดู workflow history ได้

## Team Lead

- เห็นเฉพาะงานของทีมตัวเอง
- เปิดรายละเอียด assignment ได้
- บันทึก category และแนวทางแก้ไขได้
- ส่งกลับ admin ได้
- ส่งต่อหัวหน้ากลุ่มงาน/หัวหน้างานได้พร้อมเหตุผล

## Department Head

- เห็นงานที่ส่งตรงมาหาตัวเอง
- เห็นงานเพิ่มตาม visibility ที่กำหนดไว้
- บันทึกแนวทางแก้ไขและส่งกลับทีมได้

## Director

- เปิด dashboard ได้
- เปิดรายการรายงานแบบอ่านอย่างเดียวได้
- เปิดรายละเอียดรายงานแบบอ่านอย่างเดียวได้
- ไม่มีปุ่มแก้ไขหรือส่งต่อ

## Audit / History

- Workflow History แสดงรายการย้อนหลังได้
- กรองตาม action, ผู้ดำเนินการ, วันที่ และคำค้นได้
- Export CSV ได้
- เปิด audit detail ได้

## Final Sign-off

- Master data ครบ
- User จริงครบ
- Public password ตั้งแล้ว
- Active fiscal year ถูกต้อง
- Director account พร้อม
- ผ่านการทดสอบทุก role
